So, how my final solution should look like?
Just wrap code inside function expression adn return aggregated expression. Here is the solution: 

from datetime import date


def days_diff(a: tuple[int, int, int], b: tuple[int, int, int]) -> int:

    return abs((date(*a)-date(*b)).days)
Since this solution is short and simple, you may use lambda expression (anonymous function) to write such functions more compact: 

from datetime import date


days_diff = lambda a, b: abs((date(*a)-date(*b)).days)

This mission is possible to solve without using "butteries" - module datetime, only using calculations 

def days_diff(date1, date2):

    total = []
    for date in [date1, date2]:
        year = date[0]
        month = date[1]
        a = (14 - month)//12
        y = year - a
        dayN = date[2]+(153*(month + 12*a - 3)+2)//5 + 365*y + y//4 - y//100 + y//400
        total.append(dayN)

    return abs(total[0]-total[1])
